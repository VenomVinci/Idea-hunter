# Idea Hunter — Frontend

A professional, purple-themed Next.js (App Router + TypeScript + Tailwind) frontend for Idea Hunter — an AI-powered startup idea validation platform.

## Tech
- Next.js 14 (App Router)
- TypeScript
- Tailwind CSS (custom purple brand theme)

## Getting Started

```bash
npm install
npm run dev
```

Open http://localhost:3000.

## Scripts
- `npm run dev`: Start dev server
- `npm run build`: Production build
- `npm run start`: Start production server
- `npm run lint`: Lint

## Structure
- `src/app` — routes and layout
- `src/components` — UI components
- `src/types` — shared TypeScript types
- `src/lib` — utilities
- `src/app/api/analyze` — mock analysis API

## Notes
This project ships with a mock API. Replace it with real data sources and models to provide genuine market research, competitor intelligence, and risk analysis.
